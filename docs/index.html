WeChat Desktop Skill

A reusable Codex skill for safe macOS WeChat desktop automation: detect the visible window, inspect chat context, switch chats by name, scroll history in the right pane, and send a user-approved reply.

Open README Open SKILL.md

What It Does

Uses conservative UI automation and screenshot checks instead of blind keyboard control.

Platform

Built for macOS with desktop WeChat, AppleScript, screen capture, and Swift available.

Install Shape

Ship the skills/wechat-desktop folder into ~/.codex/skills or install from GitHub.

Quick Install

mkdir -p ~/.codex/skills
cp -R skills/wechat-desktop ~/.codex/skills/

Restart Codex after installation.

Typical Prompts

  • $wechat-desktop
  • Use $wechat-desktop to switch to a WeChat chat named "test chat".
  • Use $wechat-desktop to inspect the current WeChat window and draft a reply.

Safety Model

  • Operate only inside the right message pane when reading history.
  • Prefer screenshot verification before risky actions.
  • Use a dedicated low-risk test chat for smoke tests.
  • Require macOS Accessibility and Screen Recording permissions.
